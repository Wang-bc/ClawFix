# 工具说明

当前系统可用工具如下：

## 内部工具

- `memory_search`
  用于从 Markdown 知识库、历史案例和长期记忆中检索相关内容
- `memory_get`
  用于读取指定知识文档
- `workspace_list`
  用于查看工作区目录
- `workspace_read`
  用于读取工作区文本文件
- `time_now`
  用于获取当前时间

## 外部工具

- `web_search`
  用于进行轻量 Web 搜索，返回标题与链接
- `web_fetch`
  用于抓取指定网页正文，不执行浏览器自动化

## 使用原则

1. 先内部检索，再外部补充。
2. 外部资料优先选择官方文档、规范、issue 和高质量社区总结。
3. 工具结果只作为证据，不直接替代最终分析。
